# Automated Job Application Filler

Python + Selenium script that auto-fills job applications (generic forms + LinkedIn Easy Apply).

**⚠️ Important Legal & Ethical Notice**  
Automating applications **may violate** the Terms of Service of LinkedIn and many job boards.  
Use **responsibly**, at **low volume**, for **personal/educational purposes only**. The author is **not responsible** for account bans or other consequences.

## Features
- Loads your info from `personal_info.json`
- Supports file upload (resume)
- Basic LinkedIn Easy Apply handling (phone, extra questions, multi-step forms)
- Easy to extend for other sites
