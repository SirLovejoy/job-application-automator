# Job Application Automation Helper

Simple Python + Selenium script to assist with filling out job applications — especially LinkedIn **Easy Apply**.

**⚠️ Important legal & ethical warning**  
Automating job applications **can violate** platform terms of service (including LinkedIn).  
Use **very sparingly**, **manually review** every application, and **at your own risk**.  
This is for **personal/educational purposes** only.

## Installation

```bash
# Clone (or download ZIP)
git clone https://github.com/SirLovejoy/job-application-automator
cd job-application-automator

# Recommended: virtual environment
python -m venv venv
source venv/bin/activate      # Linux/macOS
# or
venv\Scripts\activate         # Windows

pip install -r requirements.txt
